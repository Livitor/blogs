显示文件内容

```
type 文件名
```

显示目录内容

```
dir 路径
```

显示目录树

```
tree
```

创建文件

```
touch 文件名
```

删除文件

```
erase 文件名/文件夹名
```

创建文件夹

```
mkdir 文件夹名
```

同时运行多个命令

```
命令1 & 命令2 & ..
```

用户与组

```
显示用户:
net user

添加用户:
net user 用户名 密码 /add

用户添加到adminstrators组中:
net localgroup administrator 用户名 /add
```

查看共享的文件夹

```
net share
```
